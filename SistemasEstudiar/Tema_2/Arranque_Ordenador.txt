Los pasos que sigue el ordenador al arrancar:

1. Se activa la corriente
2. Se presiona el botón que permite que el ordenador se encienda -> Power Switch
3. La BIOS toma el control y hace comprobaciones
4. Comprueba elementos de hardware como:
    - Microprocesador
    - Memoria RAM
    - Tarjeta Gráfica
    - Placa Base y Alimentación
    - Teclado

5. Pueden pasar dos cosas: 

    · NO funciona -> BIOS ha encontrado un error, lo notifica por pantalla o emite un pitido o varios, dependiendo del error que haya encontrado. 
    · SI funciona -> seguimos el proceso.

6. BIOS busca los dispositivos de arranque como:
    - HDD
    - SSD 
    - USB
    - BLU-RAY
    - CD/DVD
    - Tarjeta SD
    - A través de red

7. Definido el arranque, BIOS busca el Sistema Operativo y lo carga en RAM
8. Sistema Operativo toma el control y BIOS desaparece
