Otra versión del Copyright. 

- Está hecho para regular las licencias de software libre
- Esta legislación se encarga de impedir que sea usado el código fuente sin autorización
- Define las condiciones bajo las cuales se puede hacer uso de él


    · Licencia de Software Libre sin protección Copyleft
        - Al distribuir un software bajo esta licencia, estás autorizando a terceros para no solo modificar tu producto, sino también, para poder licenciar el mismo bajo sus propios términos
        - Esto puede provocar que un software libre hecho a partir de tu obra y que estaba destinado a apoyar a la comunidad pueda ser privatizado por algún desarrollador que lo modifique

    · Licencia de Software Libre con protección Copyleft
        - Esta licencia obliga a que los programadores que quieran redistribuir tu código deban seguir los lineamientos y restricciones que les hayas impuesto.
